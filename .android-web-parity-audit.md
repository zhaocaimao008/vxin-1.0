# Android ↔ Web 功能对齐审计（2026-07-13）

对比方法：提取两端调用的后端 API 面 + 功能组件，归一化路径后取差集，再逐项人工核对。

## ✅ 已对齐（两端都有）
登录/注册/找回密码/账号切换、聊天(私聊/群聊)、消息发送/图片/语音/视频/文件、
撤回/删除/编辑/转发/引用回复、表情反应、@提及、群管理(建群/邀请/踢人/角色/转让/解散/群二维码/群头像/群公告/置顶)、
朋友圈(发布/点赞/评论/通知)、收藏、通讯录/好友请求/备注/拉黑、语音视频通话(含群通话)、
红包、会话内搜索、全局搜索、消息收藏、贴纸发送/收藏、改密/注销(本轮刚修)。

## ⚠️ 确认缺失（Web 有，Android 无）— 需补齐

### P0（核心/影响体验，优先）
| # | 功能 | Web 位置 | 后端 API | 说明 |
|---|------|---------|---------|------|
| 1 | **我的钱包**（余额/流水/充值） | Profile.jsx `Wallet` | `GET /api/wallet`、`/api/wallet/transactions`、`POST /api/wallet/recharge` | Android 完全无钱包页；红包依赖余额，缺口明显 |
| 2 | **阅后即焚**（会话消息定时焚毁） | ChatWindow.jsx | `PUT /api/messages/conversation/{id}/burn-after` | Android 无入口/无 API |
| 3 | **标为未读** | ChatList.jsx | `POST /api/messages/conversation/{id}/mark-unread` | 会话列表长按缺"标为未读" |

### P1（增强/便利）
| # | 功能 | Web 位置 | 后端 API | 说明 |
|---|------|---------|---------|------|
| 4 | **好友标签/分组** | ContactList.jsx | `/api/friend-labels/*` | Android 通讯录无标签分组 |
| 5 | **自定义表情上传** | StickerPanel.jsx | `POST /api/stickers/upload` | Android 只能发/收藏，不能上传自定义 |
| 6 | **多设备会话管理**（登录设备列表/踢下线） | Profile(sessions) | `GET/DELETE /api/auth/sessions` | Android 无"登录设备"管理 |
| 7 | **文件传输助手** | ChatWindow | `/api/messages/file-helper` | Android 无该入口 |

### 平台差异（非缺陷，无需对齐）
- Web 推送：`vapid-public-key`/`web-subscribe`（Android 用 FCM device-token，已有）。
- `/api/upload/credential`、`private/batch`、`messages/{id}/around`：Web 内部实现细节，Android 走等价路径。
