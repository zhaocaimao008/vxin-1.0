# 快速验证指南

## 现状检查

```bash
# 1. 后端状态
ps aux | grep "node.*server" | grep -v grep
# 应该看到: node /root/v信/backend-v2/src/server.js

# 2. Nginx 状态
sudo systemctl status nginx
# 应该看到: active (running)

# 3. HTTPS 证书
ls -lh /etc/nginx/certs/vxin.*
# 应该看到: vxin.crt 和 vxin.key
```

## 功能验证

### ✅ 登录 API 正常
```bash
curl -X POST http://localhost:3002/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800001111","password":"123456"}'
```
预期: 返回用户数据（登录成功）或验证错误

### ✅ 登录限制工作
```bash
# 执行 5 次失败登录（同一电话号码）
for i in {1..6}; do
  curl -X POST http://localhost:3002/api/auth/login \
    -H "Content-Type: application/json" \
    -d '{"phone":"13800001111","password":"wrong"}' \
    -w "\n[第 $i 次]\n"
done
```
预期: 第 5 次开始返回 "请稍后再试"（rate limit）

### ✅ HTTPS 工作
```bash
curl -k https://localhost/api/auth/login
# -k 忽略自签名证书警告
```
预期: 返回同 HTTP 的相同响应

### ✅ HTTPS 安全头
```bash
curl -I -k https://localhost/
# 看 Strict-Transport-Security、X-Frame-Options 等头
```
预期: 返回安全相关的响应头

## Token 黑名单验证（高级）

1. **登录并获得 token**
```bash
RESPONSE=$(curl -s -c cookies.txt \
  -X POST http://localhost:3002/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800001111","password":"123456"}')
```

2. **使用 token 访问受保护端点**
```bash
curl -s -b cookies.txt \
  http://localhost:3002/api/auth/me
# 预期: 返回用户信息
```

3. **执行 logout**
```bash
curl -s -b cookies.txt \
  -X POST http://localhost:3002/api/auth/logout
# 预期: {"success":true}
```

4. **再次尝试使用旧 token（应被拒）**
```bash
curl -s -b cookies.txt \
  http://localhost:3002/api/auth/me
# 预期: 401 "无效的Token，请重新登录" 或类似
```

## 完整测试场景

```bash
#!/bin/bash
# run_all_tests.sh

echo "=== 1. 检查后端 ==="
curl -s http://localhost:3002/api/auth/login | head -c 50
echo ""

echo "=== 2. 检查 HTTPS ==="
curl -k -s https://localhost/api/auth/login | head -c 50
echo ""

echo "=== 3. 检查登录限制 ==="
for i in {1..6}; do
  STATUS=$(curl -s -w "%{http_code}" http://localhost:3002/api/auth/login \
    -d '{"phone":"13800001111","password":"wrong"}')
  echo "请求 $i: $STATUS"
done

echo ""
echo "=== 所有检查完成 ==="
```

保存为 `run_all_tests.sh`，执行:
```bash
bash run_all_tests.sh
```

---

**验证时间**: 约 5-10 分钟  
**所需工具**: curl, sqlite3, ab (可选)
