# v信 - 微信克隆项目

## 项目结构

```
v信/
├── backend/     # Node.js 后端 (Express + Socket.io + SQLite)
├── web/         # Web 前端 (React + Vite)
├── mobile/      # 移动端 (React Native + Expo)
└── desktop/     # 桌面端 (Electron)
```

## 功能特性

- 用户注册/登录
- 添加好友（搜索、发送请求、接受/拒绝）
- 私聊 + 群聊
- 实时消息（WebSocket）
- 发送图片/文件
- 朋友圈（发布、点赞、评论）
- 个人资料编辑、头像上传
- 正在输入状态
- 在线/离线状态

## 快速启动

### 1. 安装依赖

```bash
# 后端
cd backend && npm install

# Web 端
cd ../web && npm install

# 桌面端
cd ../desktop && npm install

# 移动端
cd ../mobile && npm install
```

### 2. 启动后端

```bash
cd backend
npm run dev
# 运行在 http://localhost:3001
```

### 3. 启动 Web 端

```bash
cd web
npm run dev
# 运行在 http://localhost:3000
```

### 4. 启动桌面端（Electron）

Web 端启动后，在新终端：
```bash
cd desktop
npm install
NODE_ENV=development npx electron .
```

### 5. 启动移动端（Expo）

```bash
cd mobile
npm install
npx expo start
# 扫描二维码用 Expo Go 运行，或连接模拟器
```

> 移动端需修改 `mobile/src/config.js` 中的后端地址为本机 IP，例如 `http://192.168.1.x:3001`

## 技术栈

| 端 | 技术 |
|---|---|
| 后端 | Node.js + Express + Socket.io + SQLite |
| Web | React 18 + Vite + Socket.io Client |
| 移动 | React Native + Expo + React Navigation |
| 桌面 | Electron（加载 Web 端） |
