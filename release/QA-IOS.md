## 常见问题

**Q: CocoaPods 安装失败？**
```
sudo gem install cocoapods
pod setup
```

**Q: 证书签名错误？**
Xcode → Signing & Capabilities → 选择你的 Team，Xcode 自动管理。

**Q: Push Notification 无法使用？**
需要：
1. 在 Apple Developer 后台开启 App ID 的 Push Notification
2. 生成 APNS 证书或使用 Key
3. 配置后端 Firebase Admin SDK 或直接对接 APNS

**Q: 模拟器无法测试某些功能？**
- 相机：模拟器不支持（需真机）
- 推送：模拟器不支持（需真机）
- 后台保活：模拟器模拟有限
